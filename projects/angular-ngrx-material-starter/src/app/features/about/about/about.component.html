About PMIS

PMIS is a sytematic approach for gathering data or information regarding the Budget Execution Documents (BEDs) which contain the agency plans, spending schedules and physical targets for evaluation and consolidation. It is a tool in keeping up-dated records on the present accomplishment status of financial, physical and disbursement of the department/agency/OUs. This data/records is then processed, integrated, and stored in the centralized database and made available to all who have authority to access it, in a form that suits the department's purpose. This will be the basis for measuring the agency's performance.

System Updates

Version 5.0

  • PMIS dashboard was added for comprehensive analysis of financial, physical and disbursement per programs/OUs/Agency
  • A Graphical data was incorporated for better analysis
  • A Summary table of all programs/OUs/Agency in the Dashboard page
  • A Token Based Authentication was embedded for system security
  • An error handler was added for better capturing of bugs and http error response

Version 4.0

  • The BED-3 Component was added for disbursement plan and accomplishment monitoring
  • User's Audit Logs was incorporated in all BEDs to monitor the last activity of the responsible user.
  • Object-Code Summary Table was added in the BED-1 Component for easy verification and counter checking in the Budget records.
  • The Budget User Account can now enable to add object code to the specific activity when there's a re-alignment happening.
  • The sub-header of the activity was included in the table display for sub-grouping aggregation
